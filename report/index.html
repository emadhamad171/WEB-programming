Селектори тега - це вибірка елементів за іменами їх тегів: h1, p, img, a і т. д.

У HTML документі, селектори CSS класу знаходять елементи з потрібним класом. Атрибут класу визначається як розділений пробілами список елементів, і один із цих пунктів повинен точно відповідати імені класу, наведеному у селекторі.

У HTML-документах CSS-селектори ID виробляють вибірку всіх елементів ID, повністю збігаються з селектором.

Дочірні селектори: дочірнім називається елемент, який безпосередньо розташовується всередині батьківського елементу.

Зірочка (*) – універсальний селектор для CSS. Відповідає будь-якому тегу.
